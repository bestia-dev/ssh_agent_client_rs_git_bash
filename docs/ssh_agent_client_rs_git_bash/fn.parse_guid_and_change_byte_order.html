Parse guid and change byte order.

Original guid looks like this: 01020304-05060708-090A0B0C-0D0E0F10. Two hexadecimals digits form one u8 byte. There are 4 groups with 4 u8 bytes. Eight hexadecimal digits form one u32 byte. That is one group.
